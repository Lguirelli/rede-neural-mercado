# Neo Quant Glass Dashboard

Placeholder de plataforma para análise de 15 ações brasileiras com previsão de alta ou queda por rede neural.

## Rodar localmente

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Estrutura

- `index.html`: entrada do projeto
- `src/main.jsx`: aplicação React
- `src/styles.css`: estilo Neo Quant Glass
- `package.json`: dependências e scripts

## Observação

Os dados são simulados para visualizar layout, navegação e hierarquia de informação enquanto a rede neural real é construída.
